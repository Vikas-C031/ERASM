# Enterprise Resource Allocation & Skill Management System (ERASM)

## 📌 Project Overview

The Enterprise Resource Allocation & Skill Management System (ERASM) is a Spring Boot application designed to help IT organizations efficiently manage employees, skills, projects, and resource allocation.

## 🎯 Objectives

- Centralized employee management
- Skill inventory management
- Project management
- Resource allocation
- Approval workflow
- Utilization dashboard
- Audit logging
- Report generation
